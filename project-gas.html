Unreal Engine 5 Gameplay Ability System Framework · 2025

A modular combat framework built around Unreal Engine 5 Gameplay Ability System, focused on responsive player input, scalable ability architecture, motion matching locomotion, combat feedback, attributes, effects, damage calculation, and reusable gameplay systems.


The objective of this project was to move from isolated gameplay features to a reusable combat framework where player actions, enemy behavior, damage, UI, animation, and feedback are connected through a modular GAS-based architecture.

  • Designed and implemented the gameplay framework architecture around Unreal Engine Gameplay Ability System
  • Created native Gameplay Tags for attributes, input, damage types, weapon states, abilities, montages, effects, and gameplay events
  • Implemented a custom Ability System Component to handle ability input through Gameplay Tags
  • Built replicated Attribute Sets for health, stamina, primary attributes, secondary attributes, resistances, and combat values
  • Implemented damage flow using Gameplay Effects, SetByCaller damage values, blocked hit state, critical hit state, and floating damage feedback
  • Structured abilities such as melee attacks, dodge, guard, equip weapon, projectile abilities, and special abilities
  • Designed a motion matching locomotion system from scratch for the framework
  • Modified attack animations in Cascadeur to improve weight, timing, spacing, and combat readability
  • Used native Gameplay Tags as the communication layer between abilities, inputs, damage types, weapon states, montages, and effects
  • Mapped input tags to gameplay abilities so player input can activate abilities in a modular way
  • Created replicated attributes for Strength, Defense, Endurance, Vigor, Stamina Recovery Rate, Armor, Max Health, Max Stamina, Block Effectiveness, Critical Hit Chance, Posture Damage, Stamina Regeneration, and damage resistances
  • Extended the gameplay effect context to carry blocked hit and critical hit information through the damage pipeline
  • Connected PlayerState, AbilitySystemComponent, AttributeSet, HUD, and Widget Controllers to drive UI from gameplay data
  • Improved combat readability by editing attack animations in Cascadeur, adding more weight, clearer spacing, and better timing between key poses
