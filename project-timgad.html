Echos of
Timgad

SoulsLike Combat System · 2024 · Internship Project

A gameplay showcase from Echos of Timgad, a PC action-adventure prototype built in Unreal Engine 5 and set in the ancient Roman city of Timgad, Algeria. The combat direction was inspired by Souls-like games, with a focus on strategic timing, readable attacks, stamina management, and responsive third-person gameplay.


During my internship at Shédio Design, I worked for a period of 5 months on Echos of Timgad. My contribution focused on gameplay programming: player locomotion, vaulting, backstabing, equipment, weapon handling, combat logic, stats, targeting, hit reactions, enemy AI, and gameplay polish.

  • Implemented a modular combat component for player and enemy characters
  • Built attack combinations with input buffering for smoother combo flow
  • Developed directional dodge logic connected to combat states and stamina usage
  • Created hit detection logic for applying damage during attacks
  • Implemented health, stamina, defense, and damage-related stat systems
  • Integrated multiple weapon types with different combat behavior
  • Worked on targeting, hit reactions, combat feedback, sound cues, and visual effects
  • Implemented enemy combat AI using AI Controllers, Behavior Trees, and Blackboards
  • Improved character animation integration using Animation Blueprints, IK Rig, and IK Retargeter
  • Used Animation Blueprints and state machines to manage locomotion states such as idle, walk, run, jump, landing, and crouch
  • Used motion warping, animation montages, animation notifies, sphere traces, and components to build traversal and backstab mechanic
  • Designed a reusable equipment architecture using a base equippable actor class and base weapon actor class
  • Implemented combat foundations including combos, dodge, damage handling, hit detection, and death logic
  • Used maps for stat management, including base values, modified values, max constraints, health, stamina, armor, and defense
  • Used Behavior Trees and Blackboards to create AI decision-making, patrol, perception, chase, and attack behavior
