# Walid Kettab - Portfolio 🩸

A high-contrast, "blood red" and slate-themed professional portfolio showcasing my work as a Gameplay Programmer.

## 🌟 Overview
This repository contains the source code for my personal portfolio. It is designed to be highly immersive, featuring:
- **Cinematic Dark Theme**: Deep slate background (`#09090b`) with a striking "Blood Red" accent (`#c0392b`).
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Dynamic Interactions**: Custom cursors, glassmorphism UI elements, ambient glows, and smooth scroll animations.
- **Modern Layout**: Grid-based projects display and interactive skill badges.

## 🛠️ Tech Stack
- **HTML5**: Semantic and accessible structure.
- **Vanilla CSS3**: Custom CSS variables, flexbox/grid layouts, keyframe animations, and backdrop filters (no external CSS framework used).
- **Vanilla JavaScript**: For smooth scrolling, custom cursor logic, and mobile menu toggling.

## 🚀 Deployment
This project is optimized to be deployed seamlessly via [Vercel](https://vercel.com). Simply import the repository, and Vercel will handle the static hosting automatically.
